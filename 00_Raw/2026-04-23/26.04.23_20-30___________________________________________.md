## 3. 기술 스펙 및 자동화 워크플로우
### 3.1 이미지 생성 (Image Generation)
*   **모델:** Gemini Image Generator (hyper-realistic prompt engineering 적용)
*   **프롬프트 전략:** 'Vogue magazine style', '8k resolution', 'photorealistic cinematic lighting' 등의 키워드를 조합하여 에디토리얼 화보 감성을 극대화함.

### 3.2 이미지 호스팅 (Image Hosting)
*   **서비스:** Uguu.se (일시적 퍼블릭 URL 생성)
*   **방식:** 로컬 이미지를 API를 통해 업로드하여 Instagram Graph API가 접근 가능한 외부 URL 확보.

### 3.3 게시 자동화 (Instagram Publishing)
*   **API:** Instagram Graph API v23.0
*   **스크립트:** `insta_fashion_carousel_upload.py`
    *   `media` 엔드포인트를 통한 개별 아이템(is_carousel_item=true) 컨테이너 생성.
    *   `CAROUSEL` 타입의 상위 컨테이너 생성 및 자식(children) ID 연결.
    *   `media_publish`를 통한 최종 발행.

## 4. 성과 및 향후 전략
*   **시각적 일관성:** 모든 이미지가 한국인 여성 모델을 기반으로 일관된 톤앤매너를 유지하여 브랜드 아이덴티티 강화.
*   **확장성:** 향후 패션뿐만 아니라 라이프스타일, 인테리어 등 고관여 카테고리로 동일 포맷 확장 가능.
*   **안정성:** 웹 기반 대시보드 대신 검증된 Python CLI 스크립트를 사용하여 오류 없는 실행 확인.
