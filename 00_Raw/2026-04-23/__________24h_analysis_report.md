#### CTR 하락의 구조적 원인

**가설 ①: 썸네일 타깃 범위가 극단적으로 좁음**
- A안 썸네일은 "72시간 + 마늘 빼기" = 현재 대상포진 환자에만 즉시 반응

- 탐색 기능 유입(42.9%)이 최대인데, 이 층에게 완전히 무반응

**가설 ②: "대상포진"이라는 병명이 썸네일에 노출되면 타깃 자동 축소**
- 시니어 건강 채널의 보편 시청자는 "대상포진"을 검색할 동기가 낮음
- "대상포진 경험자"만 반응 → 국내 60대 유병률 기준 약 3~4%에 해당
- 나머지 96%+ 시청자는 진입 게이트 자체에서 차단

**가설 ③: "시간 압박(72시간)" 훅은 해당 증상자에게만 작동**
- "72시간"은 이미 진단받은 사람에게만 긴박한 숫자
- 예방 관심층에게는 오히려 거부감 ("나 안 걸렸는데 왜?")

## 4. 병목 진단

### 퍼널 분석 (24시간차 기준)

```
   노출 683
     ↓  ← [병목 1: 썸네일] CTR 0.9% (채널 기준 3~4% 미달)
   클릭 6 (노출에서 발생한 조회수)

   체류 3:16
     ↓  ← [우수 통과] 만족도 100%
   좋아요 남김
```

**병목 지점**: **썸네일 진입 단계 단 1곳**

### 병목이 "썸네일"이 아니라 "주제"인 이유

초기 진단(15시간차)에서는 "썸네일 교체로 해결"을 제안했으나, 24시간차 데이터 검토 후 **재진단**:

| 관찰 데이터 | 해석 |
|---|---|
| 노출 확장에도 클릭 안 늘어남 | 썸네일 문제 이상, 주제 흥미도 자체가 낮음 |
| 알림 유입(구독자)마저 반응 약함 | 기존 구독자층도 대상포진에 무관심 |
| YouTube 검색 유입 0% | 주제 검색 빈도 자체가 낮거나 경쟁 강자 존재 |
| AVD·좋아요는 정상 | 진입한 소수는 만족 = 콘텐츠 품질 문제 아님 |

**결론**: 
> 썸네일만 바꿔서는 한계. **주제 자체의 시장성 부족**이 근본 원인.
> 단, 썸네일 교체는 마지막 신호 송출 시도로 여전히 가치 있음 (하단 6번 참조).

## 7. 학습 포인트

### 🎓 이 영상에서 얻은 교훈 5가지 (다음 영상부터 적용)

#### 1️⃣ 대상포진 같은 "선별성 높은 주제"는 첫 영상급으로 불리

**교훈**:
- 대상포진 국내 60대 유병률 약 3~4%
- 즉, 시청 대상이 **시니어 전체의 3~4%에 불과**
- 채널이 충분히 성장한 후, 검색 유입으로 꾸준히 쌓는 **롱테일 영상**으로 적합

**다음 영상 적용**:
- 첫 영상급 주제는 **혈압·혈당·수면·피로·관절** 같은 **보편 관심사 (유병률 30%+)**
- 선별성 주제는 채널 구독자 5,000명 이상 성장 후 배치

#### 2️⃣ "시간 압박(72시간)" 훅은 해당 증상자에게만 작동

**교훈**:
- "72시간"은 이미 진단받은 사람에게만 긴박
- 예방 관심층에게는 오히려 거부감
- 시간 압박형 훅은 **"지금 이 순간" 공감할 수 있는 일상 행동**에만 사용

**다음 영상 적용**:
- "지금 드시는 혈압약"·"오늘 아침 일어났을 때"·"매일 마시는 물" 같은 **일상 앵커**
- "다음 72시간"·"남은 3일" 같은 **증상자 한정 문구 배제**

#### 3️⃣ "절대 먹지 말라"류 금지 훅은 시니어에게 역효과

**교훈**:
- 시니어 시청자는 **평생 쌓아온 식습관 신념이 강함**
- "마늘 빼세요" = 거부감 유발 (마늘은 한국인 상식상 '무조건 좋음')
- 시니어는 **"이건 꼭 드세요"** 권유형에 3~5배 높은 반응

**다음 영상 적용**:
- 제목·썸네일에 **"먹지 마세요·빼세요·끊으세요"** 단어 자제
- 대신 **"꼭 드세요·챙기세요·시작하세요"** 권유형 우선

#### 4️⃣ 썸네일에 병명 자체를 쓰면 타깃 자동 축소

**교훈**:
- "대상포진" 대신 "면역 위기", "당뇨" 대신 "혈당 경고", "고혈압" 대신 "혈관 비상"
- 병명은 의학 전문가 검색어지, 시청자 호기심 유발어가 아님

**다음 영상 적용**:
- 썸네일·제목에 **증상·신호·결과** 중심 키워드
- 예: "60대 아침 두통 3가지 신호" > "60대 기립성 저혈압 증상"

#### 5️⃣ 첫 24시간 CTR 2% 미만이면 영상 본체보다 패키징 재설계가 답

**교훈**:
- 콘텐츠는 좋아도 **패키징(썸네일+제목+설명)이 부적합하면 사장**
- 다음부터는 **제작 전** 썸네일·제목 A/B/C 3안 병렬 구상
- 업로드 직전 팀(또는 Claude)에게 **"어느 안이 가장 클릭하고 싶으세요?"** 검증

**다음 영상 적용**:
- 스크립트 완성 후 **썸네일 3안 + 제목 3안 사전 설계**
- 업로드 직전 24시간 유보 후 A안 확정 → 2시간차 CTR 확인 후 B안 대기

## 8. 다음 영상 방향 전환

### 🚀 확정된 다음 영상 주제

**"60대 아침에 일어날 때 이 증상 3가지, 혈관 비상 신호입니다"**

### 이 주제가 대상포진의 모든 약점을 보완하는 이유

| 항목 | 대상포진 편 | 아침 혈관 편 |
|---|---|---|
| 유병 범위 | 60대 중 3~4% | 60대 누구나 겪는 아침 루틴 |
| 공감도 | 환자에게만 | 전체 시니어 |
| 썸네일 단어 | "대상포진" (타깃 축소) | "아침·증상·신호" (타깃 확장) |
| 시간 압박 | "72시간" (환자 한정) | "매일 아침" (보편 일상) |
| 검색량 | 낮음 + 강자 채널 경쟁 | 높음 + 강자 분산 |
| 예상 CTR | 0.9% | 3~5% 목표 |

### 3가지 증상 구성 (제안)

1. **기상 후 두통·어지러움** — 기립성 저혈압, 뇌혈류 저하
2. **아침 손·얼굴 부종** — 야간 신장·혈관 순환 부담
3. **기상 후 가슴 답답함** — 새벽 심근경색 골든타임 징후

### 대상포진 편 교훈 반영 설계

- ✅ 썸네일에 "병명" 대신 "아침·혈관·비상" 증상 키워드
- ✅ "72시간" 대신 "매일 아침" 일상 앵커
- ✅ "빼세요" 없이 "꼭 확인하세요·꼭 드세요" 권유형만
- ✅ 본론 2~4를 각 증상 1개씩 배치 (3챕터 구조)
- ✅ v2.5 SAFE 전면 적용, 다음 영상 예고 금지, 쿠팡 링크 금지

## 9. 부록 — 채널 자산화 관점

### 이 영상은 실패가 아니라 "채널 자산"입니다

비록 CTR 0.9%로 초기 성과는 저조하지만, 이 영상은 다음 3가지 측면에서 채널의 **장기 자산**입니다:

#### 9-1. 롱테일 검색 유입 자산

대상포진 주제 특성상, 앞으로 **실제 대상포진 환자가 검색으로 유입되는 꾸준한 롱테일 영상**으로 작동합니다. 
- 초기 폭발은 없지만, 6개월~1년에 걸쳐 **조회수 누적**
- 검색 유입은 매우 타깃성이 높아 **AVD 40%+ 달성 가능성**
- 채널 구독자 획득률도 검색 유입이 홈 피드보다 2~3배 높음

#### 9-2. 컴플라이언스 자산 (v2.5 SAFE 정석 템플릿)

이 영상은 건강누림라이프 **v2.5 SAFE 원칙이 전면 적용된 최초의 풀패키지**입니다:
- ⑭ 출처 인용 시스템 (5회 이상)
- ⑮ AI 환각 차단 절차
- ⑯ 위험 표현 회피
- ⑰ 의료인 사칭 금지
- ⑱ 콜드오픈 구조 (18초, Google Flow)
- ⑲ Studio 태그 시스템 (52개)
- ⑳ 대본-장면 분리 출력
- ㉑ 독립 영상 검증

**향후 모든 영상의 품질 기준**이자 **YouTube 정책 감사 대응 자료**로 활용 가능합니다.

#### 9-3. 학습 데이터 자산

24시간 동안 축적된 실패 데이터는 **다음 영상의 성공 확률을 극적으로 높이는 학습 자산**입니다:
- 타깃 범위 설정 기준 (유병률 30%+ 주제 우선)
- 썸네일 단어 선택 원칙 (병명 배제, 증상·신호 중심)
- 훅 설계 원칙 (권유형 > 금지형)
- 초기 24시간 지표 판정 기준 (CTR 2% 미만 즉시 대응)
