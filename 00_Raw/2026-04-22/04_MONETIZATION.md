## 1. 수익화 철학

### 기본 원칙

**두 가지가 동시에 성립해야 함**:

1. 🙏 **"돈으로 값어치 매길 수 없는 가치"** (대표님의 처음 비전)
2. 💰 **"지속 가능한 수익"** (서버비, 개발비, 콘텐츠 제작비)

이 두 가지가 충돌하지 않고 **조화롭게 공존**하는 구조를 만들어야 합니다.

### 가장 위험한 함정

```
❌ "좋은 앱 만들고 나중에 수익화하자"
→ 이미 사용자가 무료에 익숙해져서 결제 거부
→ 결국 광고 도입 → 본질 훼손 → 실패

❌ "광고 한 번만 넣을까"
→ 그 순간 "광고 앱"이 됨
→ 특히 어린이 대상 앱에서 치명적

❌ "무료 기능 일부러 망가뜨려서 결제 유도"
→ 단기 수익은 올라가지만 평판 추락
→ 충성 사용자 이탈
```

### 성공하는 길

```
✅ 처음부터 후원 모델 + 광고 없음을 분명히 알리기
→ 사용자가 "이 앱은 자발적으로 지원하는 문화"로 인식
→ 충성도 높은 사용자 형성

✅ 수익보다 더 중요한 것 = 충성 사용자 1,000명
→ 그분들이 자녀, 부모, 교회에 추천
→ 자연 확산

✅ "가치가 먼저, 수익은 결과"
→ 가치 있는 앱이면 수익은 따라옴
→ 역순이면 둘 다 잃음
```

## 2. 수익 모델 5가지

### 🟢 모델 A: 자발적 후원 (Donation) ⭐ 핵심

**컨셉**: "이 앱이 도움 되셨다면, 다른 가정도 누릴 수 있도록"

#### 후원 옵션

```
💌 한 번 후원:  3,000원 / 5,000원 / 10,000원
💝 매달 응원:  3,000원 / 5,000원 / 10,000원
🌟 평생 후원자: 50,000원 (한 번)
👑 천사 후원:  100,000원+ (한 번 또는 매달)
```

#### 후원자 혜택

**모든 후원자**:
- 따뜻한 감사 메일
- 후원금 사용처 분기 리포트

**평생 후원자** (50,000원+):
- 앱 내 "감사의 벽"에 이름 등재 (선택)
- 우선 지원 이메일

**천사 후원** (100,000원+):
- 매년 감사 카드 + 로고몽 굿즈 발송
- 새 기능 베타 우선 접근
- 대표님과 직접 대화 기회 (선택)

#### UI 디자인 예시

```
┌────────────────────────────────┐
│  💝 우리집 예배를 응원해주세요    │
│                                │
│  이 앱은 광고 없이              │
│  가족의 후원으로 운영돼요         │
│                                │
│  💌 한 번만 후원       [3,000원] │
│  💝 매달 응원         [10,000원] │
│  🌟 평생 후원자        [50,000원]│
│                                │
│  *후원금은 서버비, 새 콘텐츠     │
│   제작에 100% 사용됩니다*        │
│                                │
│  [후원하기]        [나중에]      │
└────────────────────────────────┘
```

#### 예상 수익

**사용자 1만명 기준**:
- 후원율 5% 가정 (500명)
- 평균 후원금 5,000원/월
- = **월 250만원** (연 3천만원)

**벤치마킹**:
- 미국 [YouVersion 성경앱](https://www.youversion.com): 연간 기부금으로 운영 (무료, 사용자 4억명)
- 한국 컴패션, 굿네이버스: 자발적 후원 모델

**의도적으로 저렴하게**:
```

이유:
- 많은 가정이 부담 없이 접근

**성공적 전환 요소**:
1. 무료 사용자도 평생 가치 있는 경험
2. Premium 전환 유도 안 하기 (팝업 X, 제한 X)
3. 사용자가 자연스럽게 "더 원하게" 되게
4. 7일 무료 체험 → 자동 결제 (투명하게)
5. 취소 쉽게 (한 번 클릭)

#### 예상 수익

**사용자 1만명 기준**:
- Premium 전환율 3% (300명)

### 🔵 모델 C: 교회 라이선스 (B2B) ⭐ 가장 안정적

**컨셉**: "개별 가정 대신 교회 단위로 판매"

#### 가격 정책

| 교회 규모 | 월 요금 | 연 요금 |
|---|---|---|
| 🌱 작은 교회 (50가정 이하) | 30,000원 | 300,000원 |
| 🌳 중형 교회 (50~200가정) | 50,000원 | 500,000원 |
| 🌲 대형 교회 (200가정 이상) | 100,000원 | 1,000,000원 |
| ⛪ 교회 협의체·총회 | 500,000원 | 5,000,000원 |

#### 제공 기능

```
✓ 교회 로고·정체성 노출
✓ 우리 교회 공지 탭
✓ 셀별 통계 대시보드
✓ 목사님 메시지 푸시
✓ 무제한 가정 등록
✓ 교회 행사·헌금 공지
✓ 분기별 사용 리포트
✓ 우선 기술 지원 (24시간 내)
✓ 모든 가족 구성원 자동 Premium
```

#### 영업 전략

**1단계: 대표님 교회 먼저**
- 무료 또는 파일럿 가격 (할인 50%)
- 3개월 사용 후 증언 확보
- 성공 사례 영상·문서 제작

**2단계: 지인 교회 확대**
- 추천 네트워크
- 노회·교단 행사 참가
- 목회자 대상 세미나

**3단계: 교단 차원 도입**
- 주요 교단과 파트너십
- 총회 라이선스 판매
- 연간 표창 이벤트

#### 예상 수익

**한국 교회 약 8만 곳**:
- 0.1% 가입: 80교회 × 월 50,000 = **월 400만원**
- 0.5% 가입: 400교회 × 월 50,000 = **월 2,000만원**
- 1% 가입: 800교회 × 월 50,000 = **월 4,000만원**

## 3. 단계별 전략

### 🌱 1단계 (v1.0~1.5, 5월~8월): 신뢰 구축

**목표**: 충성 사용자 100명 확보

```
✅ 광고 없음 명시
✅ 모든 핵심 기능 무료
✅ 하단에 작은 "💝 후원하기" 버튼만 추가
✅ 사용자 피드백 충분히 수집
✅ 가정예배 경험담 수집 (증언)

수익 목표: 월 30~50만원
→ 주로 자발적 후원
→ 서버비 + 도메인 비용 충당
```

### 🌿 2단계 (v1.5~1.8, 8월~10월): 가치 확장

**목표**: 첫 유료 사용자 + 교회 파트너 확보

```
✅ Premium 1~2개 기능 추가 (TTS, 무제한 프로필)

✅ "프리미엄 사용자는 1가정에 1가정 후원" 메시징
✅ 첫 교회 라이선스 베타 시작 (3개 교회)
✅ 토스페이먼츠 연동

수익 목표: 월 100~200만원
- 후원: 50만원
- Premium: 50만원
- 교회: 50만원
```

### 🌳 3단계 (v2.0~2.5, 12월~2027.03): 본격 수익화

**목표**: 안정적 수익 + 자립 운영

**목표**: 비영리 재단화, 글로벌 진출

```

## 7. 윤리 가이드라인

### 절대 지킬 약속 10가지

수익화 이후에도 **타협 불가**:

#### 1. ❌ 광고 절대 없음 (특히 어린이 화면)
어떤 형태도 금지: 배너·비디오·인터스티셜·협찬 등

#### 2. ❌ 가족·아이 데이터 절대 판매·공유 안 함
마케팅사, 광고사, 제3자 판매 금지

#### 3. ❌ 결제 강요 알림 안 보냄
"Premium 가입하세요" 팝업 X, 빈도 제한 X

#### 4. ❌ 무료 기능 의도적 망가뜨리기 안 함
다른 앱들이 자주 함. 우리는 절대 안 함.

#### 5. ❌ 심리적 압박·FOMO 조장 안 함
"지금 안 사면 손해" 같은 메시지 금지

#### 6. ❌ 과도한 푸시 알림 안 보냄
하루 3개 이상 금지, 사용자 설정 존중

#### 7. ❌ 자동 갱신 숨기기 안 함
명확히 고지, 쉽게 해지

#### 8. ✅ 수익의 10% 이상 어려운 가정 환원
매년 무료 평생 라이선스 선물 (경제적 어려움 가정)

#### 9. ✅ 투명한 운영 보고서
분기별 "수익·지출·환원 보고" 공개

#### 10. ✅ 사용자 피드백 항상 존중
매달 피드백 리뷰, 분기마다 공개 답변
