## 1. 비전 선언문

### 우리가 만들고 싶은 것

**"가족이 매일 5분이라도 자연스럽게 모이게 하는,  
어린이가 평생 신앙의 등대로 삼을 수 있는 앱."**

### 우리가 절대 안 만들 것

- ❌ 광고로 도배된 앱
- ❌ 결제 강요하는 앱
- ❌ 어린이 데이터를 팔아먹는 앱
- ❌ 무료 기능 일부러 망가뜨려서 결제 유도하는 앱
- ❌ 본질을 잃고 화려한 기능만 쌓아가는 앱

### 핵심 약속 5가지

1. **광고 절대 없음** (특히 어린이 화면)
2. **가족·아이 데이터 절대 판매·공유 안 함**
3. **결제 강요 알림 안 보냄**
4. **무료 사용자도 평생 가치 있는 경험 보장**
5. **수익의 일정 비율은 어려운 가정에 무료 평생 라이선스로 환원**

### 4.1 핵심 신규 기능 A~G

#### A. 가족 프로필 시스템 ⭐ 최우선

```
┌─────────────────────┐
│ 👨‍👩‍👧‍👦 우리 가족      │
├─────────────────────┤
│ 👨 아빠 (전준호)     │ → 본인 스티커·기도
│ 👩 엄마             │ → 본인 스티커·기도
│ 👦 큰아들           │ → 본인 스티커·이야기
│ 👧 작은딸           │ → 본인 스티커·이야기
│ + 가족 추가         │
└─────────────────────┘

기능:
✓ 각자 PIN 4자리 로그인
✓ 개인 스티커·기도 분리
✓ 가족 공동 기도는 별도 섹션
✓ 어린이 모드 전환 (UI 단순화)
```

**구현 난이도**: 중간  
**예상 시간**: 2주

#### B. 365일 오늘의 말씀

```
✓ 매일 다른 구절 자동 표시 (365개 준비)
✓ 카드 수집 (1년 후 365개 모음)
✓ 좋아하는 말씀 즐겨찾기
✓ 카테고리별 검색 (위로·용기·감사 등)
✓ 월별 테마 (1월 새로운 시작, 12월 감사 등)
```

**데이터 준비**: 365개 구절 선별 (한 달 작업)

#### C. 성경 이야기 20편으로 확장

**기존 8편 + 추가 12편**:

- 구약:
  - 아담과 하와
  - 아브라함의 순종
  - 야곱의 사다리
  - 사무엘의 어린 시절
  - 엘리야와 까마귀
  - 다니엘의 친구들 (사드락·메삭·아벳느고)
  - 에스더의 용기

- 신약:
  - 베드로의 회복
  - 바울의 회심
  - 탕자의 비유
  - 착한 사마리아인의 자매 (마르다와 마리아)
  - 십자가의 사랑

**형식**:
- 동화책 톤 유지
- 각 1,500자 이내
- 핵심 성경 구절 포함
- 가족 토론 질문 3개
- 로고몽 가이드 메시지

#### D. TTS 음성 낭독

```
✓ 성경 이야기 자동 읽어주기
✓ 어린이가 글 못 읽어도 듣기 가능
✓ 한국어 자연스러운 음성
✓ 재생 속도 조절 (0.8x, 1x, 1.2x)
✓ 백그라운드 재생 지원
```

**기술**:
- Web Speech API (무료, 품질 보통)
- 또는 Google Cloud TTS (유료, 자연스러움)

#### E. 알림 시스템

```
설정:
✓ 아침 예배 알림 (기본 6:30)
✓ 저녁 예배 알림 (기본 21:00)
✓ 매주 일요일 가족 시간
✓ 응답받은 기도 1년 후 회상

기술:
- PWA Notification API
- 또는 Firebase Cloud Messaging
```

#### F. PWA 오프라인 모드

```
✓ 인터넷 없어도 앱 열림

✓ 스플래시 화면
```

#### G. 앱 아이콘·스플래시 디자인

```

✓ 로고몽 + 작은 십자가 또는 책
✓ 브랜드 컬러 (피치 오렌지)
✓ 1024×1024 PNG (앱스토어 표준)
✓ 다양한 사이즈 자동 생성
```

### 5.1 신규 기능

#### A. 어린이 색칠하기·미니게임

```
✓ 성경 인물 색칠 (다윗·예수님 등 20장)
✓ 노아의 방주 동물 짝맞추기
✓ 모세의 홍해 길찾기 미로
✓ 십계명 외우기 카드 게임
✓ 스티커 수집 연동 (게임 완료 시 +1)
```

**기술**: HTML5 Canvas + 터치 이벤트

#### B. 주간 암송 말씀 + 카드 수집

```
시스템:
✓ 매주 1구절 암송 도전
✓ 5번 듣고 5번 따라 말하기 (TTS 활용)
✓ 완료하면 카드 수집
✓ 1년 후 52장 컬렉션
✓ 가족 랭킹 (선택)

카드 디자인:
- 로고몽 + 말씀
- 특별한 날 (성탄절, 부활절 등) 한정판
```

#### C. 가족 단체 채팅 (앱 내)

```
✓ 가족만의 격리된 공간
✓ 서로 격려·기도 부탁
✓ 사진·이모지 공유
✓ 카카오톡과 별도의 "성스러운 공간"
✓ 암호화 (가족 외 접근 불가)
```

**구현**: Supabase Realtime 또는 Firebase

#### D. 월별 가정예배 리포트

```
📊 2026년 7월 우리 가족 리포트

✅ 예배 횟수: 22회
🔥 최장 연속: 11일
📖 가장 많이 본 이야기: 다윗과 골리앗 (3회)
🙏 응답받은 기도: 3건
⭐ 우리 가족 총 스티커: 87개

이번 달 가족의 기도 여정:
- 07.03 큰아들 시험 걱정 → 07.15 응답 ✨
- 07.12 할머니 건강 → 계속 기도 중

→ PDF로 저장, 가족 추억 만들기
```

#### E. 부모-자녀 토론 가이드 자동 생성

```
AI 기반 질문 생성:
```

## 7. v2.0 정식 출시

> **목표**: 2026.12.31  
> **테마**: 완전한 플랫폼

### 7.1 핵심 신규 시스템

#### A. 통합 로그인 시스템

**지원 로그인 방법**:
- 🥇 카카오톡 (한국 1순위)
- 🥈 구글
- 🥉 애플 (iOS 필수)
- 이메일 + 비밀번호 (선택)

**기술 스택 추천**:
- **Supabase Auth** (1순위): 오픈소스, 무료 5만명
- Firebase Auth (2순위): Google 의존, 안정적

#### B. 클라우드 백업·동기화

## 💎 성공을 위한 5가지 핵심 원칙

### 1️⃣ "매일 5분"의 약속을 깨지 않기

기능이 100개로 늘어나도 가정예배 자체는 항상 5분. 복잡해지면 안 씀.

### 2️⃣ 어린이가 항상 주인공

모든 새 기능은 "우리 아이가 좋아할까?"가 첫 질문.

### 3️⃣ 광고는 죽어도 안 넣음

한 번 양보하면 끝. 협상 불가능한 약속.

### 4️⃣ 하나님이 진짜 주인공

기능이 화려해질수록 본질 흐려지기 쉬움. "주님 → 가족 → 자녀" 순서 유지.

### 5️⃣ 베타 사용자 = 평생 가족

v0.9 써주신 분들이 v2.0의 가장 큰 자산.
